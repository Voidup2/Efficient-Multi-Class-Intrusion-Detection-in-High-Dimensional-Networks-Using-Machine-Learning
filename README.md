# Efficient-Multi-Class-Intrusion-Detection-in-High-Dimensional-Networks-Using-Machine-Learning
 The number of attacks carried out has increased over the years, and network security is one of the major problems associated with networks in this age. Conventional solutions are unable to cope with such complexities in a timely manner and hence the need arises for intelligent solutions which can learn to recognize malicious flow of data within network traffic. In the current study, the focus lies mainly on using machine learning algorithms for classifying network flows according to the nature of the attack. Since the dataset, i.e., CIC-IDS-2018 is extremely imbalanced, top twenty features responsible for predicting an attack are identified. Selection of relevant features from this dataset aims at reducing the size of the dataset and removing any unimportant feature from it, which will not serve our objective during training of data. Concluding from the results obtained from the above-stated research, it can be stated that combining feature selection and machine learning leads to the improvement in performance of IDSs. This will help in building efficient security solutions that will effectively counter the upcoming threats posed by cyber attacks.
